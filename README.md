# 🦺 Smart Safety Vest for Industrial Workers

An IoT-based wearable safety vest powered by **ESP32**, **LoRa**, and a **solar charging system**.

## Features
- 🌡️ Environmental Monitoring (BME280)
- 📍 GPS Tracking (NEO-6M)
- 🚶 Motion & Fall Detection (MPU6050)
- 📡 Long-range Communication (SX1278 RA-02 LoRa)
- ☀️ Solar Charging (DFRobot Solar Power Manager 5V)
- 🆘 SOS Emergency Button
- 🔋 Battery-powered wearable system

## Hardware
- ESP32 DevKit V1
- BME280
- MPU6050
- NEO-6M GPS
- SX1278 RA-02
- 5V Flexible Solar Panel
- DFRobot Solar Power Manager 5V
- 18650 Li-ion Battery
- Active Buzzer
- SOS Push Button
- On/Off Switch
