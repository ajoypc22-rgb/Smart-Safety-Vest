# Phase 01 - ESP32 Development Setup

## Objective

The objective of this phase is to set up the ESP32 development environment and verify that the board is functioning correctly before integrating any sensors.

## Hardware Used

* ESP32 DevKit V1
* USB Data Cable
* Computer/Laptop

## Software Used

* Arduino IDE
* ESP32 Board Package

## Tasks Completed

* Arduino IDE Installed
* ESP32 Board Package Installed
* USB Driver Installed
* ESP32 Connected Successfully
* Blink LED Test Passed
* Serial Monitor Test Passed

## Files

* ESP32_Blink.ino
* Serial_Test.ino

## Result

The ESP32 board was successfully configured and tested. The board is ready for sensor integration in the next phase.
