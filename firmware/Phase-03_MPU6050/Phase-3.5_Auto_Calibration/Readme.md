# Phase 03.5 - Automatic Reference Calibration

## Objective
Automatically calibrate the worker's normal standing posture using MPU6050.

## Features
- 5-second countdown
- 100-sample automatic calibration
- Reference Pitch calculation
- Reference Roll calculation
- Real-time Tilt Pitch
- Real-time Tilt Roll
- Moving Average Filter

## Result
The system successfully stores the worker's normal posture and continuously calculates tilt relative to the calibrated reference.
